## Description

This frontend is a simple web view developed in [ReactJS](https://pt-br.reactjs.org/), to consume the API https://todoappjsm.herokuapp.com/
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
